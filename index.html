Politique de confidentialité

Dernière mise à jour : Février 2026

1. Données collectées

LookyBody ne collecte aucune donnée personnelle. Toutes les photos et données restent stockées localement sur votre appareil. Aucune information n'est envoyée à des serveurs externes.

2. Stockage des photos

Les photos prises avec LookyBody sont stockées dans un répertoire sécurisé de l'application, séparé de la galerie de votre téléphone. Elles ne sont accessibles qu'à travers l'application, protégée par votre code PIN.

3. Code PIN

Votre code PIN est stocké de manière sécurisée grâce au système de stockage chiffré de votre appareil (Keychain sur iOS, Keystore sur Android). Il n'est jamais transmis ni stocké en clair.

4. Permissions utilisées

Caméra : Pour prendre des photos de progression corporelle.
Galerie photos : Uniquement pour importer/exporter des photos, à votre demande explicite.

Aucune permission n'est utilisée en arrière-plan.

5. Partage de données

LookyBody ne partage aucune donnée avec des tiers. Aucun outil d'analyse, de publicité ou de suivi n'est intégré dans l'application.

6. Suppression des données

Vous pouvez supprimer toutes vos données à tout moment depuis les Réglages de l'application. La désinstallation de l'application supprime également toutes les données associées.

7. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter à : contact@andraseduardgabriel.com
